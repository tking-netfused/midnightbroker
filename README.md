# MidnightBroker

MidnightBroker is a retail-focused World of Warcraft addon that provides a standalone, movable on-screen display for key player stats along with optional LibDataBroker data source output. It displays a wide range of information including current date and time, zone and subzone, player coordinates, durability, total addon memory usage, gold, frame rate, and latency, with additional broker metrics for gold, FPS, and latency. Every standalone element is independently configurable, supporting enable/disable toggling, movable positioning when unlocked, persistent position saving across reloads, and full style customization including font, font size, text color, background color, border color, alpha, and scale, as well as optional label and title visibility per element. The addon also features graceful fallback when LibDataBroker-1.1 is not available.
